# CO2 Phase Calculator iOS App

Native SwiftUI version of the CO2 phase calculator.

## Open in Xcode

1. Open `CO2PhaseCalculator.xcodeproj` in Xcode 15 or newer.
2. Select the `CO2PhaseCalculator` scheme.
3. Pick an iPhone simulator or a connected iPhone.
4. Press Run.

The app is iPhone-only, uses SwiftUI and Canvas, and runs calculations locally without network access.

## Notes

- Deployment target: iOS 17.0
- Bundle ID: `com.wcjfghk98.co2phasecalculator`
- App icons are placeholder asset slots. Add icon PNGs in Xcode before App Store distribution.
